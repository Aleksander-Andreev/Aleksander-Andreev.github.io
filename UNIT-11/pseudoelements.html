Домашнее задание про псевдоэлементы

Псевдоэлемент в CSS — это ключевое слово, добавляемое к селектору, которое позволяет стилизовать определённую часть выбранного элемента. Например, псевдоэлемент ::first-line может быть использован для изменения шрифта первой строки абзаца.

Псевдокласс в CSS — это ключевое слово, добавленное к селектору, которое определяет его особое состояние. Например, Hello wolrd! может быть использован для изменения цвета кнопки при наведении курсора на неё.

  • :hover
  • :active
  • :focus
  • :checked
  • :first-child
  • :last-child
