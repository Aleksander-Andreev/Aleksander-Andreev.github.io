Виды текстовых тегов

p-Контейнер для большого текста с абзацами.

strong-важный текст,
b - выделить жирным
i - курсив,
supверхний регистр, sub - нижний регистр,
del - удаленный текст,
ins - добавленный текст,
small - уменьшает размер текста,
code - отображает содержимое текста в стиле, предназначенном для указания на то, что текст является коротким фрагментом компьютерного кода. 

pre - представляет собой предварительно отформатированный текст, 
        который должен быть представлен точно так, как написано в 
        HTML-файле.
